# OptiSecure

**AI-Powered Security Portfolio Optimization Platform**

OptiSecure helps CISOs and security leaders optimize their security tool portfolios, eliminate redundancy, quantify risk-adjusted ROI, and make data-driven buying decisions.

## Features

- **Portfolio Overview** — Complete visibility into your security tool inventory, spend, and utilization
- **Redundancy Analysis** — Identify overlapping capabilities across 500+ vendor tools with quantified redundancy scores
- **Compliance Mapping** — Automated gap analysis against NIST CSF, ISO 27001, SOC 2, PCI DSS, and HIPAA
- **ROI & Risk Modeling** — Simulate budget scenarios and quantify marginal risk reduction per dollar spent
- **Budget Optimizer** — AI-driven recommendations to consolidate tools and cut redundant spend by 18–30%

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Recharts (data visualization)
- Lucide React (icons)
- React Router v7

## Getting Started

```bash
npm install
npm run dev
```
